colors grid

lite plugin to store your favourite colors and copy it with one click

setup

  1. include colorsGrid javascript file
    2.   <script src="colorsGrid.min.js"></script>
  2. set html container 
      <div class="container"></div>
  3. style grid items with css useing color-holder class 
    .color-holder {
  width: 200px;
  height: 200px;
  margin: 5px;
}
  4. call colorsGrid with options
    5. colorGrid(document.getElementById('container'), {
  gridItems: {
    0: "#19aad1"                    
  }
})
